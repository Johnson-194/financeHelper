PROJECT INFO:
 FinanceHelper is a program built (by me) mainly for students (anyone can use it), that lets users track their spending in a clear and easy way.
 The main purose of this program is to provide a history of your transactions (given by the user) and a summary of this including graphs.

REGARDING THE BRANCHES:
 Edits to READMEs and reports will be committed via the main branch as I know these will work and not interfere with project code.
 New branches will be made when developing parts of the program.
